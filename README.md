#  Мой путь разработчика (My Dev Journey)
## Обзор проекта
Этот репозиторий служит портфолио и хранилищем для всех моих проектов в рамках обучения веб-разработке на платформе **The Odin Project**. Здесь вы можете отследить мой прогресс от первых шагов в программировании, до выполнения сложных профессиональных задач.
## Использованные технологии
* **HTML5**
* **CSS3**
* **JavaScript** (по мере изучения курса)
* **Linux** (Ubuntu)
* **Git** & **GitHub**
* **Терминал Bash**
## Статус обучения 
На данный момент я активно прохожу учебную программу. Новые проекты и обновления добавляются регулярно.
## Как запустить проекты локально
1. Клонируйте репозиторий: `git clone github.com` 
2. Перейдите в папку проекта: `cd main-projects`
3. Откройте файл `index.html` в вашем браузере
